# Changelog

All notable changes to this project will be documented in this file.

## [0.5.0] - 2026-05-26

### Added
- Cost Optimizer module (CostTracker, CostOptimizer)
- Real-time cost tracking for OpenAI, Anthropic, Bedrock
- Cost per agent and cost per task metrics
- Budget alerts and monitoring
- Optimization suggestions engine

### Changed
- Improved __init__.py imports with safe fallbacks
- Enhanced documentation

## [0.4.0] - 2026-05-25

### Added
- Orchestration module (FintechSREOrchestrator)
- Real-time alerting (AlertManager)
- Remediation engine (RemediationEngine)
- Progressive autonomy constraints
- Prometheus metrics exporter

## [0.3.0] - 2026-05-24

### Added
- Semantic SLI monitoring (DQR, TIE, HER, AQDD)
- AgentSLICollector for task tracking
- CloudWatch integration
- A2A semantic validator
- Agent chain circuit breaker
