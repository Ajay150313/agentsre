# Contributing

Want to help build this? We could use it.

## Getting Started

```bash
git clone https://github.com/Ajay150313/agentsre.git
cd agentsre
python -m venv venv
source venv/bin/activate
pip install -e ".[dev]"
pytest
```

## What Helps

Bug fixes. New SLI ideas. Framework integrations (we need LangChain, CrewAI, more Bedrock examples). Support for GCP and Azure.

Better docs. Examples of how you're using it. Real-world scenarios.

## Making Changes

1. Fork it
2. Create a branch: git checkout -b feature/your-idea
3. Make changes
4. Add tests if you're adding code
5. Run: pytest
6. Commit and push
7. Open a PR

## Code Style

Black for formatting. Keep it simple.

## Questions?

GitHub discussions or issues. We're pretty responsive.

Thanks.
